# Examples

This page provides example resource instances demonstrating the use of Care FHIR profiles.

## Patient Example

See [Patient-care-patient-example.json](Patient-care-patient-example.html) for a complete example of a Patient resource using the Care Patient Profile with all extensions.

## Encounter Example

See [Encounter-care-encounter-example.json](Encounter-care-encounter-example.html) for an example of an Encounter resource.

## Observation Example

See [Observation-care-observation-example.json](Observation-care-observation-example.html) for an example of an Observation resource with vital signs.

## Using Examples

These examples demonstrate:
- Proper use of Care-specific extensions
- Correct profile references in meta.profile
- Appropriate value set bindings
- Resource relationships and references

When implementing Care FHIR resources, use these examples as a reference for the expected structure and content.
